# 🛍️ Retail Sales Analytics Dashboard

> End-to-End Data Analytics Project using **MySQL** & **Power BI**

---

## 📌 Business Problem

Retail businesses generate thousands of transactions every day, making it difficult to monitor sales performance, identify profitable products, analyze customer behavior, and track regional trends. This project transforms raw retail data into an interactive dashboard for data-driven decision making.

---

## 📊 Project Summary

- 📁 **5** Relational Tables
- 📈 **40+** Business SQL Queries
- 📊 **1** Interactive Power BI Dashboard
- 📌 **8+** Business KPIs
- 📉 Sales Trend Analysis
- 👥 Customer Analysis
- 📦 Product Performance Analysis
- 🌍 Regional Performance Analysis
- 🔄 Return Analysis
